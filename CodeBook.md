##Prerequisits:

1. Zip unpacked to ~/UCI, ~/UCI/test, ~/UCI/train2
2. Processing script loaded to  ~

##Processing steps

1. Load and prepare activity ids and names
2. Load, select and prepare valid feature names (only mean, std)
3. Read test data
4. In the test data frame: select columns containing valid features
5. Read training data
6. In the training data frame: select columns containing valid features
7. Merge test and training data frames 
8. Add additional columns: subject and activity
9. Name columns
10. Replace activity ids with activity names
11. Sort data frame
12. Establish groups by subject and activity. Apply colMeans to the each group
13. Split the grouping column 
13. Write the output data frame to a file
